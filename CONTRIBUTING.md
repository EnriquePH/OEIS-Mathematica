# Contributing

Contributions are welcome.

## How to contribute

1. Fork the repository.
2. Create a branch for your change.
3. Make your changes and test them with Wolfram Language when possible.
4. Submit a pull request with a clear description of the improvement.

## Guidelines

- Keep the code readable and documented.
- Prefer small, focused changes.
- Preserve backwards compatibility when possible.
- Mention any relevant tests or examples.
